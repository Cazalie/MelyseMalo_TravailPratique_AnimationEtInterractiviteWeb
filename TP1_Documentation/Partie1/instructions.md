# TP / Volet #1 - Page fil de fer

## Objectif

Créer une **page « fil de fer »** qui fait la promotion du jeu que vous développez dans le cours 2J2 (jeu 2D avec Unity). Cette page doit refléter fidèlement votre jeu avec du contenu réaliste et pertinent.

Votre page doit être optimisée pour **mobile en premier** mais tout en s'adaptant pour les fenêtres larges.

## Éléments requis

1. **Navigation** adaptative complète et fonctionnelle
2. **Tout le contenu** demandé dans la scénarisation fournie, bien hiérarchisé
3. **Un ajout personnel** à la scénarisation (créativité bienvenue !)
4. **Une galerie** des jeux développés par les étudiant·es du cours `2J2` (une grille de *cartes*)
5. **Pied de page** avec votre propre contenu

## Consignes importantes

- **L'utilisation de tout modèle IA est absolument interdite à ce stade-ci.**
- **Ne pas utiliser de vraies images**, seulement des zones grises qui peuvent être marquées d'un x
- **Seulement les palettes monochromes sont acceptées.** Le gris est encouragé
- **Le lorem ipsum est interdit**, le texte doit être intelligible et pertinent
- **Toutes les tailles de fenêtre entre 420px et 1800px sont supportées.** En aucun cas, l'interface devrait-elle *briser* entre ces tailles
- **Seuls les fichiers nécessaires au fonctionnement doivent être remis.** La présence de tout autre fichier sera considéré comme un manque de professionnalisme
- **Des transitions fluides doivent être appliquées**, là où vous jugez qu'elles contribuent à l'expérience utilisateur. Un minimum de transitions courantes est attendu
- **L'imbrication CSS devrait être utilisée** pour rassembler les styles interdépendants
- **Les propriétés CSS personnalisées doivent être utilisées** pour le langage graphique
- **Les requêtes média doivent être utilisées** lorsque l'adaptation fluide est impossible
